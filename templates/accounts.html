PDF X-RAY is split into a few different models, so that users can choose which feature-set best works for them. Accounts come in four different forms: anonymous, standard, premium and premium local. Below is a breakdown of the features included for each type of account (each account includes the previous accounts features):


Anonymous Accounts (free):

  • Ability to upload samples using the web interface

  • Ability to view uploaded sample report and manual analysis

  • Access to view the last 25 reports uploaded

  • Ability to view how users rated streams

  • Ability to search based on various hashes

  • Ability to request expert analysis on uploaded files

Standard Accounts (free):

  • Access to a limited API for submissions and report viewing

  • Ability to see the last 50 reports uploaded

  • Account page that tracks uploaded files

  • Ability to flag streams as malicious or not

  • Access to limited training module

Premium Accounts:

  • Access to the full API

  • Access to more powerful searching including object contents, suspicious details and more

  • Access to thousands of malicious reports

  • Access to related stream detection module

  • Subscription to malicious file/components hash list

  • Priority processing on uploads

  • Access to more extensive training module

Local Premium Accounts:

  • Local installation of PDF X-RAY on site
